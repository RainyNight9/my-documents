## 9、vue3新特性

1、更快

    * 虚拟DOM重写
    * 优化slots的生成
    * 静态树提升（内存换时间）
    * 静态属性提升 
    * 基于Proxy的响应式系统

2、更小

    * 通过摇树优化核心库体积

3、更容易维护

    * TS+模块化

4、更加友好

    * 跨平台：编译器核心和运行时核心与平台无关

5、更容易使用

    * 改进ts支持，编译器提供更好的类型检查和错误及警告
    * 更好的调试支持
    * 独立的响应式模块
    * Composition API
